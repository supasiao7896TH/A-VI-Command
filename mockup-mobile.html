📋 สิ่งที่จะปรับปรุง / เพิ่มใหม่

📱 PWA Installation

  • เพิ่ม manifest.json พร้อม icons ทุกขนาด
  • Service Worker สำหรับ cache assets offline
  • Install prompt สำหรับ Add to Home Screen
  • Splash screen ขณะโหลด App
  • Theme color เข้ากับ System UI
  • Standalone display mode

🧭 Bottom Navigation Bar

  • แทน sidebar ด้วย Bottom Nav 5 ปุ่ม
  • Active state + animation
  • Badge notification (unread chat)
  • Safe area inset สำหรับ punch-hole screen
  • Haptic feedback on tap (Web Vibration API)

🃏 Agent Grid (Mobile)

  • 3 คอลัมน์ แทน 4 คอลัมน์เดิม
  • Chief card แสดงเต็มแถว (featured)
  • Compact card: avatar + ชื่อ + แผนก
  • Tap card → ไปหน้าแชทกับ Agent นั้นเลย
  • FAB "💬" เพื่อเปิดแชทเร็ว

💬 Chat Interface

  • Horizontal scroll pill เพื่อสลับ Agent
  • Back button กลับหน้าแรก
  • Keyboard-aware layout (วิ่งขึ้นเมื่อ keyboard เปิด)
  • ไมค์ปุ่ม (เผื่อ voice input อนาคต)
  • Attachment button
  • Bubble UI ชัดเจน user vs agent

📊 Dashboard

  • Summary metric cards แบบ 2 คอลัมน์
  • Portfolio list แบบ swipeable cards
  • Tab: ภาพรวม / พอร์ต / ปันผล / Watchlist
  • Pull-to-refresh สำหรับ update ราคา
  • FAB ➕ เพิ่มหุ้นใหม่

⚡ Performance & UX

  • Safe area insets (notch / punch-hole)
  • Touch targets ≥ 44px ทุกปุ่ม
  • Prevent double-tap zoom
  • Overscroll behavior: none (ป้องกัน pull-to-dismiss)
  • System font fallback สำหรับ Thai
  • Dark mode ตาม System preference
